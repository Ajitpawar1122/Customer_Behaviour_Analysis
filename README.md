Data analysis project showcasing Customer Behaviour Analysis by using Python,Sql and Power Bi.


📊 Data Analytics Project – End-to-End Data Analysis Pipeline

This project demonstrates a complete Data Analytics workflow, starting from loading a dataset in Python to generating business insights through SQL, Power BI dashboards, and a professional report. The project follows industry-standard data analytics practices, making it suitable for portfolios, recruiters, and interviews.
